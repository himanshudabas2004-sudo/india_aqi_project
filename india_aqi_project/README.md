# India Air Quality (AQI) Analysis Dashboard

This project analyzes India city-level air quality trends and provides an interactive dashboard.

## Features
- Daily AQI trends by city
- AQI category breakdown
- Cross-city comparison for any selected year
- Pollutant indicators including PM2.5, PM10, NO2, SO2, and CO

## Run locally
```bash
pip install -r requirements.txt
python dashboard/app.py
```

Then open the dashboard at `http://127.0.0.1:8050`.
